What is Sql Server?

Sql Server is a database manager used by most companies/developers.

It can be accessed/manipulated by code in order to be used in applications/web services.

It's one of the most advanced technolgies on the market today.

Common Sql Server Commands

Command Description
SELECT {column(s)} Displays designated columns in a table.
INSERT {table} set Inserts a new row in {table}
DELETE {column} from {table} Deletes a {column} from the {table}
JOIN {table} ON {table1PK} = {table2FK} Joins 2 tables where one PK matches another FK
