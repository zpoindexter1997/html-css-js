What is Git?

Git is the local controller to alter and configure repositories.

It can be linked to GitHub so that your local reposity and online repository can speak with each other.

You can also communicate with other online repositories, as long as they are public.

Common Git Commands

Command Description
cd {directory} Moves you into designated directory
mkdir {name} Creates a new folder in your current folder
.git push {branch} Pushes commits of {branch} to repository
.git clone {url} Creates a clone of {url} repository in your current folder
